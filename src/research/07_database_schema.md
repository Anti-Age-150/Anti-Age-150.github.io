# Database Schema

| Field | Description |
|---------|---------|
| Category | Research area |
| Subfield | Specialized topic |
| Person / Org | Researcher or company |
| Institution | Affiliation |
| Tenure / Role | Academic or company role |
| Key Contribution | Major impact |
| Key Papers | Landmark work |
| Translational Status | Research to clinic |
| Evidence Level | A-E |
| Notes | Miscellaneous |
| Links | References |

## Evidence Levels

A = Human clinical outcomes
B = Human biomarker evidence
C = Strong animal evidence
D = Cell / in vitro
E = Hypothesis or commercial claim
